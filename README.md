# 🏋️ Gym Website Clone

> A modern, responsive gym website built using HTML, CSS, and JavaScript.
> Designed to replicate a professional fitness brand landing page with clean UI, responsive layout, and smooth interactivity.

---

## 📖 Overview

This project is a front-end clone of a professional gym/fitness website.
It demonstrates strong fundamentals of **semantic HTML**, **modern CSS styling**, and **vanilla JavaScript interactivity**.

The goal of this project was to practice:

* Responsive web design
* UI/UX layout structuring
* DOM manipulation
* Clean folder architecture
* Real-world landing page development
